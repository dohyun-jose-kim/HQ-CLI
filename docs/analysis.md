# 분석 결과

## CLI 출력 예시

이 CLI는 `DataAnalysis_vf.ipynb`의 HQ_2(개인별 보충제 + 그룹 평균 식이)에 상응하는 계산을 수행한다.

![CLI 출력 예시](../assets/example_result.png)

### 해석 방법

- 영양소는 HQ 값 기준 **내림차순**으로 나열된다.
- 영양소명은 데이터의 **표준화된 컬럼명**으로 표시된다.
- **빨간색 막대**: HQ >= 1 — 상한섭취량을 초과하여 주의가 필요한 영양소
- **초록색 막대**: HQ < 1 — 해당 입력자의 성별/연령을 반영했을 때 안전한 수치
- 빨간 점선은 HQ = 1 (상한) 기준선이다.

### 참고 사항

- **보충제 섭취량**은 사용자가 직접 입력한 개인화된 값이다.
- **식이 섭취량(MHI)** 은 해당 성별/연령 그룹의 평균값을 사용한다. 개인의 실제 식이와 차이가 있을 수 있다.

## 결과 저장 위치

| 파일 | 경로 | 설명 |
|------|------|------|
| HQ 그래프 | `results/HQ_(timestamp).png` | 실행마다 개별 생성 |
| 누적 결과 | `results/result_accumulation.xlsx` | 실행마다 1행씩 누적 |
| 오류 보고서 | `results/report_accumulation.csv` | 사용자 피드백 누적 |

## 수정 가이드

결과 분석이나 시각화를 수정/확장하고 싶을 때:

| 변경 내용 | 수정 대상 |
|-----------|-----------|
| 그래프 스타일, 색상, 레이아웃 변경 | `result_maker.py` |
| HQ 계산 로직 변경 | `data_processor.py` |
| 누적 결과 컬럼/포맷 변경 | `main.py` (`save_result_to_excel`) |
| 결과 출력 메시지 변경 | `main.py` (`main` 함수 내 Phase 5) |
